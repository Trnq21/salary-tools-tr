 Salary Tools TR (v2)

Đây là phiên bản mới của web Salary Tools TR:

- Card 1: Tính lương Gross → Net theo tháng.
- Card 2: Quyết toán thuế năm theo từng tháng, có:
  - Nhập lương gross và thuế TNCN đã khấu trừ theo từng tháng.
  - Tính bảo hiểm, khấu trừ khác, giảm trừ bản thân & người phụ thuộc cả năm.
  - Tính thuế phải nộp năm và chênh lệch với thuế đã khấu trừ.

## Cách dùng

1. Đổi tên file này thành `index.html` (nếu bạn chỉ tải riêng file).
2. Đưa `index.html` lên hosting (GitHub Pages / Netlify / Vercel).
3. Nếu bạn đã có repo `salary-tools-tr`:
   - Chỉ cần thay thế file `index.html` cũ bằng file mới này và commit lại.
